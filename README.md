# 地震學與板塊構造 · 第五章學習指南（原創中文版）

互動式教學網頁：板塊運動學、擴張中心、隱沒帶、板內地震、斷層與變形。

- **線上閱讀（GitHub Pages）**：https://oceanicdayi.github.io/Seismology_ch5/
- 內容為依據 Stein & Wysession, *An Introduction to Seismology, Earthquakes, and Earth Structure*, Ch.5 概念撰寫之**原創摘要與說明**；不含原書文字與圖片之重製，所有圖解為自製 SVG / Canvas 示意圖。
- 功能：各節重點摘要、白話說明與舉例、5 個互動實驗（Euler 速率計算、岩石圈冷卻、熱參數、Mohr 圓、黏滑模型）、原創練習題附詳解、10 題綜合測驗。

純靜態單檔（`index.html`），無建置流程。
